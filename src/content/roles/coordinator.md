## When This Role Is Needed

Coordinator is needed whenever a squad requires operational support.

## Typical Requests

- Create a repository
- Grant access
- Create a project folder
- Configure GitHub Projects
- Share links to project resources

## How To Start

Introduce yourself in the community and ask which squad needs operational support.
