## When This Role Is Needed

When a project needs technical direction, review quality, or production reliability.

## Typical Requests

- Review a pull request
- Propose an ADR or RFC
- Define repository standards

## How To Start

Open issues labeled for contributors, review existing PRs, and introduce your technical focus.
